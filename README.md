# raspberry_pi_nas_ui

Nowoczesny interfejs użytkownika do zarządzania serwerem Raspberry Pi NAS.

## Funkcje
- Monitorowanie parametrów systemowych (CPU, RAM, Temperatura, Dysk) w czasie rzeczywistym.
- Podstawowe sterowanie systemem (Restart, Wyłączenie, Aktualizacja).
- Responsywny design dostosowany do urządzeń mobilnych.
